Install react js follow step
step 1:- download node setup (https://nodejs.org/en/)
step 2:- install node setup
step 3:- check it will install properly or not with help of cmd --> open cmd and type node -v -->showing node 	version(eg v16.10.0)
step 4:- type npm install -g Create-react-app in cmd(eg C:\WINDOWS\System32> npm install -g create-react-app)
step 5:- create react app --> goto folder with help of cmd in cmd where you can create react application --> and 	type create-react-app app_name (eg. create-react-app demo program)
step 6:- showing Happy hacking! msg so its means application created

Run application step-->
	go to folder with help of cmd and type simply npm install it will be run..
	(eg. D:\akshay\React\project\demo>npm start
